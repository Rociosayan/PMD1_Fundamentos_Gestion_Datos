# Laboratorios FGD

Esta carpeta agrupa los laboratorios creados para Fundamentos de Gestion de Datos.

## Estructura

- `calificados/`: laboratorios calificados `LAB-C1` a `LAB-C8`, en formato Word y/o notebook cuando existe.
- `dirigidos/`: laboratorios dirigidos `LAB-D1` a `LAB-D15`, en formato Word y/o notebook cuando existe.
- `recursos/`: bases SQLite e imagenes de apoyo usadas por los laboratorios.

No se incluyen scripts generadores, plantillas, archivos temporales, copias de respaldo ni carpetas de trabajo.
